# 📦 Packet Capture

## Overview

Packet capture is the process of collecting network traffic for analysis. It helps understand how devices communicate over a network.
---

## Steps

1. Open Wireshark.
2. Select the active network interface.
3. Click **Start Capturing Packets**.
4. Generate network traffic by opening a website or using the `ping` command.
5. Stop the capture and review the collected packets.
---
## Why Packet Capture?

- Observe network communication.
- Troubleshoot connectivity issues.
- Analyze network protocols.
- Build practical networking skills.
---
## Summary

Packet capture is the starting point for network traffic analysis in Wireshark.
