# 🌍 DNS Analysis

## Overview

DNS converts domain names into IP addresses. Wireshark allows us to inspect DNS requests and responses.

---

## Display Filter

```text
dns
```

---

## What to Observe

- Query Name
- Response
- IP Address
- Response Time

---

## Why DNS Analysis?

- Verify domain resolution.
- Detect unusual DNS activity.
- Troubleshoot DNS-related issues.

---

## Summary

DNS analysis helps understand how domain names are resolved before a connection is established.
