# 🔍 Display Filters

## Overview

Display filters allow you to focus on specific packets in a capture, making analysis faster and more organized.

---

## Common Filters

| Filter                  | Description                                   |
|--------                 |-------------                                  |                                  
| tcp                     | Show TCP packets                              |
| udp                     | Show UDP packets                              |
| dns                     | Show DNS traffic                              |
| http                    | Show HTTP packets                             |
| https                   | Show HTTPS traffic                            |
| icmp                    | Show ICMP packets                             |
| ip.addr == 192.168.1.10 | Show packets related to a specific IP address |

---

## Why Use Display Filters?

- Reduce unnecessary traffic.
- Quickly locate relevant packets.
- Simplify troubleshooting and analysis.

---

## Summary

Display filters are one of the most useful features of Wireshark for analyzing network traffic efficiently.
