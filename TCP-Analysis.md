# 🌐 TCP Analysis

## Overview

TCP (Transmission Control Protocol) provides reliable communication between devices. In Wireshark, TCP packets can be analyzed to understand how connections are established and data is transferred.

---
## Common Observations
- Three-way handshake
- Sequence numbers
- Acknowledgements
- Connection termination

---

## Display Filter

```text
tcp
```

---

## Why TCP Analysis?

- Understand reliable communication.
- Troubleshoot connection problems.
- Analyze network performance.

---

## Summary

TCP analysis helps identify how devices communicate reliably across a network.
